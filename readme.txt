Materials were used in the template:

1. Icons Font Awesome (Fonts are in the folder - fonts/fontawesome-webfont),
(Font connection styles are in the folder - css/font-awesome.min.css).
2. Fonts (https://fonts.googleapis.com/css?family=Roboto:400,700&amp;subset=cyrillic),
(https://fonts.googleapis.com/css?family=Courgette&amp;subset=latin-ext).
3. Bootstrap is an open source toolkit for developing with HTML, CSS, and JS
(Bootstrap styles are in css/bootstrap.min.css),
(https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css),
(https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/js/bootstrap.min.js),
(js/jquery-3.2.1.slim.min.js).
4. Slick slider (Slick styles are in slick/slick.css), (slick/slick-theme.css).
5. jQuery library was used. jQuery is a fast, small, and feature-rich JavaScript library 
(js/jquery-3.2.1.slim.min.js), (https://code.jquery.com/jquery-3.3.1.slim.min.js).
6. Popper.js (https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js)
7. Parallax (js/parallax.min.js).
8. Images used from the site (https://pixabay.com/en/).
